# TriVision Firmware

This firmware demonstrates sensor integration for the TriVision Smart Driver Monitoring Board.

## Features

* ESP32-S3 based firmware
* MAX30102 heart-rate sensor interface
* MPU6050 motion sensing
* Alert indication through buzzer
* Green status LED
* Red warning LED
* Serial monitoring support

## Future Enhancements

* BPM calculation
* Driver fatigue detection
* Bluetooth connectivity
* Mobile application integration
* TinyML-based driver behavior analysis
* Camera-based drowsiness detection

## Platform

* ESP32-S3
* Arduino IDE / PlatformIO
* I2C Communication
